# Python Face Detection & recognition Module (Personal)

> A Python Script that Detect faces form target image and compare against known faces and extract the unknown match 

## Usage: 

```python
usage: face_rec.py [-h] [--extract-unknown] --target-dir TARGETDIR [--gen-has] [--known-has DAT] [--out OUT] [--tol TOL] [--frame FRAME]

  optional arguments:
  -h, --help            show this help message and exit
  --extract-unknown     Extract the Unknown faces to the Local disk
  --target-dir TARGETDIR
                        Target Directory to Compare Faces / Generate HAS
  --gen-has             Generates file of known faces
  --known-has DAT       Holds a Known face database
  --out OUT             Collection for Output / Name of the Output
  --tol TOL             Custom Tolerence Value Default is EXT for extraction
                        Other options CMP for comparison
  --frame FRAME         Custom Frame size default is 30
```
